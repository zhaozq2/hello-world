## Snake Game

A classic Snake game implemented using:
- HTML5 Canvas for rendering
- JavaScript for game logic
- CSS3 for styling

### How to Play

1. Open `snake-game.html` in your browser
2. Click "Start Game"
3. Use Arrow Keys or WASD to control the snake
4. Eat red food to grow and earn points
5. Avoid walls and yourself

### Features

- Score tracking
- Increasing difficulty
- Pause/Resume functionality
- Responsive design
- Keyboard controls (Arrow Keys + WASD)

## Getting Started

Clone this repository and open `snake-game.html` in a web browser to play the game.
