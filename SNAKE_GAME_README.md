# Snake Game

A classic Snake game implemented in HTML5, CSS3, and JavaScript.

## How to Play

1. Open `snake-game.html` in your web browser
2. Click the "Start Game" button
3. Use **Arrow Keys** or **WASD** to control the snake
4. Eat the red food to grow and earn points
5. Avoid hitting the walls or yourself

## Features

- **Responsive Design**: Works on desktop and mobile devices
- **Score System**: Earn 10 points for each food eaten
- **Increasing Difficulty**: Game speed increases as your score goes up
- **Pause/Resume**: Pause the game anytime
- **Game Controls**: 
  - Start Game: Begin a new game
  - Pause/Resume: Toggle game pause state
  - Reset Game: Restart the game

## Game Rules

- Each food gives you **10 points**
- The snake grows by one segment when it eats food
- Game ends if the snake hits the wall or itself
- Game speed increases every 50 points
